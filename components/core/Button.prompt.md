Spaced-caps editorial button; use "outline" for most CTAs, "filled" (clay) for the single primary action on a page, "link" for inline arrow-links.

```jsx
<Button variant="outline" href="/resume">Resume →</Button>
<Button variant="filled">Get in touch</Button>
<Button variant="link">See all photos →</Button>
```

Hover: outline inverts to ink; filled darkens to clay-deep; link shifts to pine.
